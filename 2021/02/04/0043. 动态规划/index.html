5. 力扣406：根据身高重建队列

思路：

  • 从高位往地位排序
  • 高个子看不到矮个子，所以根据前面有多少人比它高来插入即可

5. 力扣64：最小路径和

思路：

  • 除了第一行与第一列，其他位置的值，均是从当前位置的上方或者左方中，取一个最小值

6. 力扣647：回文子串

思路：

  • 马拉车算法
  • 通过马拉车算法得出新子串，算出半径，直接把半径 / 2的结果相加，就是最终结果

7. 力扣62：不同路径

思路：

  • 动态规划：
    • 第一行跟第一列初始化为1
    • 其他位置为当前位置的上方+左方

8.力扣279：完全平方数

思路：

  • 平方数最小为1，每个dp初始化为1的数量
  • 平方数最大不会超过目标值
  • 因为不限数量，转换为无限背包问题

10.力扣139：单词拆分

11. 力扣416：分割等和子集

思路：

  • 如果总和为奇数，则不能分割
  • 如果最大的元素 > sum / 2，则不能分割
  • dp一维数组，01背包，每次输入一个元素，只要dp[j] || dp[j - nums[i]] || nums[i] == j有一个为true，则d[j] = true

12. 力扣300：最长递增子序列

思路：

  • 方式一：以 i 结尾，当前的最长递增子序列
  • 方式二：维护一个lens数组，下标对应长度，值对应当前长度的最大值

13. 力扣221：最大正方形

思路：

  • 从左上角到右下角移动

  • 第一行与第一列初始化

  • 其他的位置，为上方、左方、左上方的最小值 + 1

15. 力扣494：目标和

思路：

  • 如果最大值小于目标绝对值，返回0
  • 因为可加可减，所以结果事存在负数的，整个dp二维数组向右边平移，保证下标不会出现负数
  • dp的状态转移：
    • 当前dp[i][j] = dp[i - 1][j - nums[i]] + dp[i - 1][j + nums[i]];

16. 力扣322：零钱兑换

思路：

  • 之前的dp，是以零钱种类为外部循环，这里反过来，以零钱种类为内部循环，当所有的价格大于零钱值是，判断dp进行转移dp[i] = min(dp[i], dp[i - coins[j]] + 1);

17. 力扣55：跳跃游戏

思路：

  • 维护一个变量right，为当前位置最远能到的位置
  • 当位置小于这个坐标的时候，不断更新right，取最大值
  • 只要right超过目标位置，即成功

18. 力扣152：乘积最大子数组

思路：

  • 维护一个最大值与最小值
  • 最终结果为当前值*最大值、当前值、当前值*最小值三者中取最大
